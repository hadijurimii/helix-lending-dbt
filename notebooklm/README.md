# NotebookLM Study Pack

Use this folder as your prep cockpit.

- `01_source_pack.md`: upload this to NotebookLM first. It is the compact source of truth.
- `02_study_guide.md`: read this when you want to understand the project end to end.
- `03_interview_battle_cards.md`: use this for likely interview questions and sharp answers.
- `04_presenter_script.md`: rehearse from this before presenting.
- `05_slide_speaker_notes.md`: slide-by-slide intent and talking points.
- `helix_lending_pipeline_walkthrough.pptx`: editable presentation deck.

Recommended NotebookLM prompts:

1. "Explain this project like I am presenting it to a senior data engineering interviewer."
2. "Quiz me on the weakest design decisions in this pipeline."
3. "Generate follow-up questions an interviewer might ask after each slide."
4. "Turn the data quality section into a concise spoken answer."
5. "Challenge the architecture choice. What would a skeptical reviewer ask?"
